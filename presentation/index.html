Eine kurze Geschichte des Webs

  • Entwickelt am CERN (Schweiz)
  • Tim Berners-Lee
  • Beginn in 1989
  • Computer-Kommunikationssystem
  • Stellt Information nicht nur bereit, sondern verlinkt sie
  • Beginn eines Informationsnetzes

Eine kurze Geschichte von HTML (1)

  • Entwickelt am CERN (ja, auch)
  • Tim Berners-Lee
  • 1980: Erstes Hypertext-System "Enquire" für den Privatgebrauch
  • Späte 1980er: Hypercard für Macintosh
  • 1990: Berners-Lee entwickelt Browser und Server
  • Am CERN nie offiziell adoptiert

Eine kurze Geschichte von HTML (2)

  • 1991: Offene Diskussionen über HTML im Internet
  • 1992: Zwei Kopien eines freien Browsers werden heruntergeladen
  • 1993: Unternehmen halten das Web für "akademisch"
  • 1994: Erste WWW-Konferenz in Genf, W3C entsteht, Gründung von Netscape
  • 1995: MS Internet Explorer
